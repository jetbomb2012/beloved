## 使用規範

1. 所有專案均使用 Gitea 管理。
2. 不再使用 Apache + CGI Git Server。
3. 每個專案建立獨立 Repository。
4. Commit 前請確認變更內容，避免提交測試檔案或暫存檔。
5. Repository 重要結構不得任意修改。
6. 每次重大架構變更須同步更新 `memory.md`。
